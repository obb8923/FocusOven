# 제빵사 레벨 및 경험치 안내

이 문서는 FocusOven 앱에서 사용되는 제빵사 레벨 시스템과 경험치(경험 포인트, XP) 규칙, 레벨별 해금 빵 목록을 정리한 자료입니다.

## 1. 경험치 기본 규칙
- 타이머가 종료되어 빵을 획득할 때마다 제빵사는 경험치를 얻습니다.
- 획득 경험치는 **집중한 시간(분)** 에 비례합니다.
  - 계산식: `XP = max(1, round(집중 시간(초) / 60))`
  - 예시: 10분 집중 → `round(600 / 60) = 10 XP`
  - 아주 짧은 세션도 최소 1 XP는 지급됩니다.
- 하루 동안 여러 번 타이머를 끝내면 누적 경험치가 계속 증가하며, 레벨업 즉시 새로운 빵이 해금됩니다.

## 2. 레벨 기준 (LEVEL_THRESHOLDS)
`src/shared/store/bakerStore.ts`에서 관리하는 레벨 기준은 다음과 같습니다.

| 제빵사 레벨 | 필요 누적 XP (이상) | 설명 |
|-------------|--------------------|------|
| 1           | 0                  | 기본 레벨. 레벨 1 빵 선택 가능 |
| 2           | 10                 | 레벨 2 빵 해금 |
| 3           | 25                 | 레벨 3 빵 해금 (현재 최대 레벨) |

> 참고: `LEVEL_THRESHOLDS` 배열은 `[0, 10, 25]`이며, 추후 레벨이 늘어나면 이 배열을 확장하면 됩니다.

## 3. 레벨별 해금 빵 목록
`src/shared/constant/breads.ts`에 정의된 빵과 레벨 관계는 다음과 같습니다.

### 레벨 1 해금 빵 (기본)
- PlainBread (식빵)
- Crouton (크루통)
- DinnerRoll (모닝빵)
- Muffin (머핀)
- Scone (스콘)
- ChocoChipCookie (초콜릿 칩 쿠키)

### 레벨 2 해금 빵 (XP ≥ 10)
- Baguette (바게트)
- Bagel (베이글)
- Pretzel (프레첼)
- CreamBread (크림빵)

### 레벨 3 해금 빵 (XP ≥ 25)
- Croissant (크루아상)
- Brioche (브리오슈)

## 4. 집중 기록 (Focus Logs)
- 타이머가 끝날 때마다 `FocusLog` 항목이 생성되어, 획득한 빵과 집중 시간, 완료 시각을 기록합니다.
- 기록은 최대 100개까지 저장되며, `FOCUS_LOGS` 키로 AsyncStorage에 보관합니다.
- ETC 화면에서 최근 기록을 확인할 수 있습니다.

## 5. 상태 저장 키 요약
- `BAKER_PROGRESS`: 경험치, 현재 선택된 빵, 보유 빵 개수를 저장
- `FOCUS_LOGS`: 집중 기록 리스트 저장

## 6. 향후 확장 시 참고 사항
- 레벨 기준을 변경하거나 레벨을 추가하려면 `LEVEL_THRESHOLDS`와 빵 데이터의 `level` 값을 함께 조정해야 합니다.
- 경험치 지급 방식(XP = 빵 레벨)을 바꾸고 싶다면 `useBakerStore` 내 `awardBread` 함수의 `xpGain` 계산을 수정하세요.
